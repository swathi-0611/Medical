# 🏥 Medical Management System

A modern web-based application to manage healthcare services including patients, appointments, and medical records in an efficient and user-friendly way.

🔗 **Live Demo:** https://medical-delta-ten.vercel.app/  

---

## ✨ Features

- Patient registration and management  
- Appointment scheduling system  
- Doctor/admin dashboard  
- Medical records tracking  
- Secure authentication  
- Responsive design for all devices
